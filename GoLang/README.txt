GET /message: Этот эндпоинт возвращает следующий вопрос из сообщений, связанных с сотрудником. Ваше расширение может отправить GET-запрос на этот эндпоинт, чтобы получить вопрос от сервера.

POST /ws: Этот эндпоинт обрабатывает WebSocket-соединение от расширения. Расширение может отправлять команды через WebSocket для сохранения ответов, приостановки и возобновления диалога.

GET /download: Этот эндпоинт обрабатывает запросы на загрузку файлов с ответами. Расширение может отправить GET-запрос на этот эндпоинт, чтобы скачать файл с историей ответов.
